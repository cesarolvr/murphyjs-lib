# MurphyJS

A JavaScript vanilla library to scroll based reveal animations.

MurphyJS is a lightweight JavaScript animation library with a simple implementation way. All this works by joining of data-attributes, Web animate API and Intersection Observer API.

## Quick Start

### Installation

```bash
npm install murphyjs
```

### Basic Usage

Just do three steps:

1. Tag your HTML with `data-murphy`:
```html
<div data-murphy="left-to-right">Any content here</div>
```

2. Reset your CSS:
```css
*[data-murphy] {
  opacity: 0;
}
```

3. Start MurphyJS:
```javascript
import murphy from "murphyjs";
murphy.play()
```
